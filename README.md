# Icog_labs_task1 Project

## Overview

This project focuses on understanding entropy and cross-entropy in machine learning using the Fashion-MNIST dataset. A simple neural network model was trained using TensorFlow/Keras to classify clothing images.

## Main Tasks

* Train a simple classification model
* Get softmax prediction probabilities
* Implement entropy manually
* Implement cross-entropy manually
* Compare correct vs incorrect predictions
* Visualize entropy and loss values

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

## Key Learning

* Entropy measures prediction uncertainty
* Cross-entropy measures prediction error
* Correct predictions usually have lower entropy and lower loss
* Incorrect predictions usually have higher entropy and higher loss

## Dataset

Fashion-MNIST dataset:

* 60,000 training images
* 10,000 testing images
* 10 clothing classes

## Conclusion

This project helped me understand how neural networks make predictions and how entropy and cross-entropy are used to measure confidence and model performance in machine learning.
